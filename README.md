# React Localization Comparison

Сравнение двух подходов к локализации на русском языке в React приложениях.

## О проекте

Это приложение демонстрирует два разных подхода к обработке русской локализации в React с TypeScript:

1. **react-i18next** - полнофункциональный фреймворк интернационализации со встроенной поддержкой плюрализации
2. **russian-nouns-js** - специализированная библиотека для склонения русских существительных по падежам

## Возможности

- **Автоматическая плюрализация**: Оба подхода автоматически обрабатывают сложные правила русской плюрализации без ручных условных конструкций
- **Согласование по роду**: Демонстрация согласования глаголов с родом автора
- **Интерактивное сравнение**: Интерфейс с двумя колонками для прямого сравнения результатов
- **Динамические элементы управления**: Слайдеры и переключатели для изменения значений в реальном времени

## Тестовые сценарии

Приложение демонстрирует три категории русского текста:

| Сценарий | Грамматическая особенность |
|----------|---------------------------|
| Восстановление статей | Плюрализация существительных с числительными |
| Авторство | Согласование глаголов по роду |

## Сравнение подходов

### react-i18next

**Преимущества:**
- Встроенная поддержка плюрализации через суффиксы (_one, _few, _many)
- Внешние JSON файлы для переводов
- Широкая экосистема и большое комьюнити
- Поддержка множества языков

**Использование:**
```typescript
t('restore_articles', { count: articleCount })
```

### russian-nouns-js

**Преимущества:**
- Специализированная поддержка всех 7 падежей русского языка
- Точный контроль над склонением
- Меньший размер бандла для русско-ориентированных приложений

**Использование:**
```typescript
const noun = getNounWithCount(articleLemma, count);
const text = `Восстановить ${count} ${noun}?`;
```

## Правила русской плюрализации

Приложение корректно обрабатывает сложные правила русской плюрализации:

| Окончание числа | Падеж | Пример |
|----------------|-------|--------|
| Оканчивается на 1 (кроме 11) | Именительный единственного | 1 статья, 21 статья |
| Оканчивается на 2-4 (кроме 12-14) | Родительный единственного | 2 статьи, 3 статьи, 4 статьи |
| Оканчивается на 0, 5-9, 11-14 | Родительный множественного | 5 статей, 11 статей, 12 статей |
