# Quick Start Guide

## Быстрый старт

### 1. Проверьте, что сервер запущен

Сервер разработки уже запущен на http://localhost:5173

### 2. Откройте приложение

Откройте браузер и перейдите по адресу: **http://localhost:5173**

### 3. Протестируйте функциональность

#### Тест плюрализации статей
1. Найдите слайдер "Количество статей"
2. Переместите его на значение **1** → Должно показать "1 статью"
3. Переместите на **2** → Должно показать "2 статьи"
4. Переместите на **5** → Должно показать "5 из 11 статей"
5. Переместите на **11** → Должно показать "11 из 11 статей"

**Проверка**: Оба столбца (react-i18next и russian-nouns-js) должны показывать идентичный текст.

#### Тест плюрализации ответов
1. Найдите слайдер "Количество ответов"
2. Переместите его на значение **1** → "1 ответ"
3. Переместите на **2** → "2 ответа"
4. Переместите на **5** → "5 ответов"
5. Переместите на **12** → "12 ответов"

**Проверка**: Оба столбца должны совпадать.

#### Тест согласования по роду
1. Найдите переключатель "Пол автора"
2. Выберите "Мужской" → Должно показать "Создал Андрей Савельев..."
3. Выберите "Женский" → Должно показать "Создала Андрей Савельев..."

**Проверка**: Глагол должен измениться в обоих столбцах одинаково.

## Troubleshooting

### Порт уже занят
Если порт 5173 занят, Vite автоматически выберет другой порт (5174, 5175 и т.д.)

### Ошибки компиляции
Проверьте консоль браузера (F12) и терминал на наличие ошибок

### Изменения не применяются
Vite использует Hot Module Replacement (HMR), изменения применяются автоматически.
Если что-то не работает, обновите страницу (F5).
